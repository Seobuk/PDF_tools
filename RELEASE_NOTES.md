# PDF Composer 릴리즈 노트

## v1.5.1 (2026-07-23)

### 버그 수정
- Windows 실행 파일이 `Could not load PyInstaller's embedded PKG archive from the executable` 오류로 실행되지 않던 문제를 수정했습니다.
  - 단일 `PDF_Tools.exe`(`--onefile`) 대신 폴더형(`--onedir`) 빌드를 `PDF_Tools-windows.zip`으로 배포합니다. 실행 데이터가 exe 끝에 파묻히지 않아 백신 오탐·다운로드 손상으로 인한 실행 실패가 사라지고, 시작 속도도 빨라집니다.
  - 사용법: zip 압축을 풀고 `PDF_Tools` 폴더 안의 `PDF_Tools.exe`를 실행하세요. exe는 같은 폴더의 `_internal`과 함께 두어야 합니다.

## v1.5.0 (2026-07-23)

### 새로운 기능
- 리눅스용 실행 파일(`PDF_Tools-linux`) 릴리즈 추가 — 다운로드 후 `chmod +x PDF_Tools-linux`로 실행합니다.

## v1.4.0 (2026-07-04)

### 새로운 기능
- PDF 병합 탭에 '파일 추가' 버튼 추가 — 드래그 앤 드롭 없이도 파일을 선택해 추가할 수 있습니다.
- 병합 지원 이미지 형식 확대: 기존 JPG/PNG에 더해 BMP, TIFF, WEBP, GIF를 지원합니다.

### 버그 수정
- 휴대폰 사진 등 EXIF 회전 정보가 있는 이미지가 눕혀진 채 병합되던 문제를 수정했습니다.
- A4 변환이 성공해도 항상 '오류 발생'이 표시되고 변환 후 미리보기가 갱신되지 않던 문제를 수정했습니다.
- PDF 분할 탭의 '선택한 범위 분할' 버튼이 활성화되지 않던 문제를 수정했습니다.
- 팔레트 모드 PNG 등 RGB 외 형식의 이미지 병합 시 발생할 수 있던 변환 오류를 수정했습니다.

### 개선사항
- PDF 회전 탭 성능 개선: 페이지 선택·회전 시 문서 전체를 다시 렌더링하지 않고 해당 페이지만 갱신합니다.
- 미리보기 공통 모듈을 도입해 탭 간 중복 코드를 제거했습니다.

## v1.3.1 (2026-07-03)

### 변경 사항
- v1.3.0 디자인에서 탭 글자가 잘려 보이는 문제로, 디자인을 v1.1 시절의 기본(Fusion) 스타일로 되돌렸습니다.
- 커스텀 테마·전역 스타일시트를 제거해 어떤 OS/DPI 환경에서도 기본 위젯 모양으로 안정적으로 렌더링됩니다.
- v1.2.1의 JPEG 변환 버그 수정 등 기능 개선은 모두 유지됩니다.

## v1.3.0 (2026-07-03)

### 디자인 개편
- 시맨틱 컬러 토큰 기반의 새 디자인 시스템 적용
  - 알약(pill) 형태의 탭 내비게이션과 흰색 카드 패널
  - 버튼 위계 구분(기본/완료/보조)과 hover·pressed·disabled·focus 상태 스타일
  - 본문 텍스트 대비 4.5:1 이상, 8px 스페이싱 리듬, 슬림 스크롤바
- PDF 회전 탭의 페이지 미리보기가 카드형으로 개선되고 선택 상태가 더 명확해졌습니다.

### 개선사항
- 전역 스타일시트 도입으로 모든 탭의 목록·입력·체크박스·툴팁 스타일이 통일되었습니다.

## v1.2.1 (2026-07-03)

### 버그 수정
- PDF → 이미지 변환에서 JPEG 형식 선택 시 항상 "이미지 변환 중 오류" 메시지와 함께 저장이 실패하던 문제 수정

## v1.1.0 (2024-04-11)

### 새로운 기능
- PDF A4 포매팅 기능 추가
  - PDF 문서를 A4 용지 크기에 맞게 자동 조정
  - 드래그 앤 드롭으로 파일 추가 지원
  - 사용자 지정 저장 위치 지원
 - PDF 미리보기 확대/축소 기능 추가

### 개선사항
- PDF 회전 기능 성능 개선
- PDF 분할 기능 안정성 향상
- 이미지 추출 품질 개선

### 버그 수정
- PDF to JPEG 변환 시 일부 페이지 누락 문제 해결
- 수식 추출 시 특수 문자 인식 오류 수정
- 메모리 누수 문제 해결

### 시스템 요구사항
- Python 3.8 이상
- PyQt5
- PyPDF2
- fitz (PyMuPDF)

### 설치 방법
```bash
pip install -r requirements.txt
```

### 알려진 이슈
- 일부 특수 문자를 포함한 PDF 문서에서 수식 추출 시 오류가 발생할 수 있습니다.
- 매우 큰 PDF 파일(100MB 이상) 처리 시 성능 저하가 발생할 수 있습니다.

### 이전 버전에서 업데이트하는 방법
1. 기존 설치된 버전 제거
2. 새로운 버전 설치
3. 설정 파일 백업 후 복원 (필요한 경우)
